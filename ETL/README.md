# ETL Assignment

This was completed for the 2nd course in the Clinical Data Science Specialization: [Clinical Data Models and Data Quality Assessments](https://www.coursera.org/learn/clinical-data-models-and-data-quality-assessments?specialization=clinical-data-science). The assignment was to perform an ETL process from the MIMIC-III database into the [OMOP](https://www.ohdsi.org/data-standardization/the-common-data-model/) Condition_Occurence table.
